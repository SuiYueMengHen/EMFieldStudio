# 高级功能说明

## 概述

DiffractionLab 提供多种专业分析工具，帮助您深入理解衍射现象和光学系统性能。这些功能位于独立的子界面中，不会影响主界面的操作。

## 参数扫描

### 功能说明

参数扫描功能允许您在指定范围内自动变化某个参数，观察衍射图样和关键指标（Strehl 比、FWHM）的变化趋势。

### 扫描参数

| 参数 | 说明 | 单位 | 物理意义 |
|------|------|------|---------|
| 波长 | 扫描入射光波长 | nm | 改变波长直接影响衍射角和衍射斑大小 |
| 孔径尺寸 | 扫描孔径有效直径 | μm | 改变孔径大小影响衍射斑大小和分辨率 |
| 传播距离 | 扫描观察面距离 | m | 改变距离影响衍射图样从近场到远场的演化 |

### 使用方法

1. 从菜单 **仿真 → 参数扫描** 打开对话框
2. 选择要扫描的参数
3. 设置起始值、结束值和步数
4. 点击"开始扫描"
5. 扫描过程中实时显示当前衍射图样
6. 完成后查看 Strehl 比和 FWHM 曲线

### 指标说明

#### Strehl 比

Strehl 比定义为实际峰值强度与理想衍射极限峰值强度之比：

$$S = \frac{I_{\text{actual}}}{I_{\text{ideal}}}$$

完美系统 Strehl = 1.0。根据 Maréchal 判据，Strehl > 0.8 的系统被认为是衍射极限系统。Strehl 比与波前 RMS 误差 $\sigma$ 的近似关系为：

$$S \approx e^{-(2\pi\sigma)^2}$$

其中 $\sigma$ 的单位为波长 $\lambda$。

**参数扫描中的 Strehl 比含义**：
- 扫描波长时：Strehl 反映不同波长下系统的聚焦质量
- 扫描孔径时：Strehl 反映孔径大小对衍射极限的影响
- 扫描距离时：Strehl 反映离焦量对峰值强度的影响

#### FWHM（半高全宽）

FWHM 是衍射斑强度下降到峰值一半时的宽度，反映系统的分辨能力：

$$\text{FWHM} = 2\sqrt{2\ln 2} \cdot \sigma \approx 2.355\sigma \quad \text{(Gaussian 近似)}$$

对于圆形孔径的艾里斑，FWHM 约为 $0.514 \lambda/D$（弧度）。

### 数据导出

扫描完成后可点击"导出数据"将结果保存为 CSV 文件，包含每步的参数值、Strehl 比、FWHM 和峰值强度。

## 曲线拟合

### 功能说明

对衍射图样的截面数据进行数学拟合，提取关键参数并评估拟合质量。拟合使用当前主界面衍射结果的水平截面数据。

### 拟合模型

#### Gaussian 拟合

$$I(x) = A \cdot \exp\left(-\frac{(x-x_0)^2}{2\sigma^2}\right) + I_0$$

参数说明：
- $A$ — 振幅，峰值的最大高度
- $x_0$ — 中心位置，峰值所在的坐标
- $\sigma$ — 标准差，控制峰的宽度
- $I_0$ — 偏移量，背景强度

FWHM 与 $\sigma$ 的关系：

$$\text{FWHM} = 2\sqrt{2\ln 2} \cdot \sigma \approx 2.355\sigma$$

**适用场景**：近似分析衍射斑宽度，快速提取 FWHM。

#### Airy 拟合

$$I(x) = A \cdot \left[\frac{2J_1(\pi x / s)}{\pi x / s}\right]^2 + I_0$$

参数说明：
- $A$ — 振幅
- $s$ — 尺度参数，控制衍射斑的宽度
- $I_0$ — 偏移量

其中 $J_1$ 为一阶贝塞尔函数。Airy 函数是圆形孔径夫琅和费衍射的理论模型。

**适用场景**：圆形孔径的精确拟合，提取艾里斑参数。

#### 双峰 Gaussian 拟合

$$I(x) = A_1 \cdot \exp\left(-\frac{(x-x_1)^2}{2\sigma_1^2}\right) + A_2 \cdot \exp\left(-\frac{(x-x_2)^2}{2\sigma_2^2}\right) + I_0$$

参数说明：
- $A_1, A_2$ — 两个峰的振幅
- $x_1, x_2$ — 两个峰的中心位置
- $\sigma_1, \sigma_2$ — 两个峰的标准差
- $I_0$ — 偏移量

**适用场景**：双缝干涉等具有两个峰值的情况。

### 拟合质量评价

R²（决定系数）衡量拟合曲线与数据的吻合程度：

$$R^2 = 1 - \frac{\sum_i(y_i - \hat{y}_i)^2}{\sum_i(y_i - \bar{y})^2}$$

- $R^2 = 1$：完美拟合
- $R^2 > 0.95$：优秀拟合
- $R^2 > 0.8$：良好拟合
- $R^2 < 0.5$：拟合不佳，考虑更换模型

### 使用方法

1. 先在主界面进行衍射计算
2. 从菜单 **分析 → 曲线拟合** 打开对话框（自动载入水平截面数据）
3. 选择拟合模型
4. 点击"执行拟合"
5. 查看拟合参数和曲线对比
6. 可导出拟合数据为 CSV

## 波前分析

### 功能说明

波前分析工具允许您通过调整 Zernike 多项式系数来构建和分析波前像差，并实时查看波前相位图和干涉图模拟。

### Zernike 多项式

Zernike 多项式是定义在单位圆上的正交多项式组，广泛用于描述光学系统的波前像差。其极坐标形式为：

$$Z_n^m(\rho, \theta) = R_n^{|m|}(\rho) \cdot \begin{cases} \cos(m\theta) & m \geq 0 \\ \sin(|m|\theta) & m < 0 \end{cases}$$

径向多项式：

$$R_n^{|m|}(\rho) = \sum_{k=0}^{(n-|m|)/2} \frac{(-1)^k (n-k)!}{k! \left(\frac{n+|m|}{2}-k\right)! \left(\frac{n-|m|}{2}-k\right)!} \rho^{n-2k}$$

**注意**：本程序使用 OSA/ANSI 标准的非归一化 Zernike 多项式。系数的单位为波长 λ，但不能直接解释为 RMS 波前误差。

### 支持的 Zernike 项

| 系数 | 名称 | 阶数 (n,m) | 物理意义 | 对像质的影响 |
|------|------|-----------|---------|------------|
| Z₀ | 平移 (Piston) | (0,0) | 常数相位偏移 | 不影响像质，仅改变总相位 |
| Z₁ | 倾斜X (Tip) | (1,1) | X方向波前倾斜 | 像点横向偏移，不降低分辨率 |
| Z₂ | 倾斜Y (Tilt) | (1,-1) | Y方向波前倾斜 | 像点横向偏移，不降低分辨率 |
| Z₄ | 离焦 (Defocus) | (2,0) | 波前二次方偏移 | 像面离焦，衍射斑扩大 |
| Z₅ | 像散0° | (2,2) | 0°方向像散 | 不同方向焦距不同，产生椭圆斑 |
| Z₆ | 像散45° | (2,-2) | 45°方向像散 | 同上，方向旋转45° |
| Z₇ | 彗差X | (3,1) | X方向彗差 | 点源呈彗星状拖尾 |
| Z₈ | 彗差Y | (3,-1) | Y方向彗差 | 同上，方向旋转90° |
| Z₉ | 三叶X | (3,3) | 三重对称像差 | 三角形对称的波前畸变 |
| Z₁₀ | 三叶Y | (3,-3) | 三重对称像差 | 同上，方向旋转30° |
| Z₁₁ | 球差 | (4,0) | 球面像差 | 边缘与中心焦点不同，无法同时清晰 |
| Z₁₂ | 二级像散0° | (4,2) | 高阶像散 | 高阶像散效应 |
| Z₁₃ | 二级像散45° | (4,-2) | 高阶像散 | 高阶像散效应 |
| Z₁₄ | 四叶X | (4,4) | 四重对称像差 | 四角星形畸变 |
| Z₁₅ | 四叶Y | (4,-4) | 四重对称像差 | 同上，方向旋转22.5° |

### 干涉图模拟

干涉图是波前与参考平面波干涉的结果。强度分布为：

$$I(x,y) = \frac{1}{2}\left[1 + \cos\left(2\pi \cdot (W(x,y) + T \cdot x)\right)\right]$$

其中：
- $W(x,y)$ — 波前像差（单位：波长 λ）
- $T$ — 参考倾斜条纹数

参考倾斜的作用是产生载波条纹，使波前畸变表现为条纹弯曲。条纹数越多，越容易观察细微的波前变化，但条纹间距也越密。

### 使用方法

1. 从菜单 **仿真 → 波前分析** 打开对话框
2. 调整各 Zernike 系数（单位：波长 λ）
3. 实时观察波前相位图和干涉图变化
4. 点击"重置所有系数"恢复初始状态

## 传播动画

### 功能说明

传播动画功能展示衍射图样随传播距离的变化过程，帮助您直观理解近场到远场的衍射演化。

### 衍射场演化

随着传播距离 $z$ 的增加，衍射图样从近场的菲涅尔衍射逐渐过渡到远场的夫琅和费衍射。菲涅尔数：

$$N_F = \frac{a^2}{\lambda z}$$

其中 $a$ 为孔径半径。

- $N_F \gg 1$：近场（菲涅尔衍射），图样类似孔径形状，边缘有衍射条纹
- $N_F \approx 1$：过渡区域，图样开始呈现傅里叶变换特征
- $N_F \ll 1$：远场（夫琅和费衍射），图样为孔径函数的傅里叶变换

### 参数说明

| 参数 | 说明 | 单位 | 建议值 |
|------|------|------|--------|
| 起始距离 | 动画开始的传播距离 | m | 设为近场距离 |
| 结束距离 | 动画结束的传播距离 | m | 设为远场距离 |
| 帧数 | 动画总帧数 | - | 20-50 帧 |
| 速度 | 每帧显示时间 | ms | 50-200 ms |

### 使用方法

1. 从菜单 **仿真 → 传播动画** 打开对话框
2. 设置起始距离和结束距离
3. 设置帧数（建议 20-50 帧）
4. 点击"生成动画"（后台计算，可随时取消）
5. 使用播放控制：
   - **播放/暂停**：自动播放动画
   - **前进/后退**：逐帧查看
   - **速度滑块**：调节播放速度（20-500 ms/帧）
   - **帧滑块**：拖动到任意帧

### 注意事项

- 帧生成在后台线程中执行，不会阻塞主界面
- **必须使用菲涅尔传播模型**（角谱法或脉冲响应），因为夫琅和费模型与距离无关
- 大帧数或大网格会导致计算时间较长
- 建议网格大小不超过 512 以保证流畅

## 接收面形状分析

### 功能说明

接收面形状分析功能模拟衍射光场投射到不同形状接收面（探测器）上的效果。实际光学系统中，探测器不一定总是平面——可能是倾斜的、弯曲的或安装在曲面基底上。本功能帮助您理解接收面形状对衍射图样的影响。

### 原理

当衍射光场投射到非平面接收面时，接收面上不同位置到孔径的有效传播距离不同。对于面型高度函数 $h(x,y)$，有效传播距离为：

$$z_{\text{eff}}(x,y) = z_0 - h(x,y)$$

其中 $z_0$ 为名义传播距离。由于衍射图样随传播距离变化，接收面上不同位置将呈现不同离焦程度的衍射图样。

程序采用**多深度切片法**实现：
1. 计算有效距离范围 $[z_{\min}, z_{\max}]$
2. 在该范围内均匀采样 $N$ 个深度切片
3. 对每个切片计算衍射强度
4. 对每个像素，根据其有效距离在相邻切片间线性插值

### 支持的接收面形状

| 面型 | 高度函数 $h(x,y)$ | 参数 | 说明 |
|------|-------------------|------|------|
| 平面 | $h = 0$ | 无 | 标准平面，无变化 |
| 倾斜(X) | $h = x \cdot \tan\theta$ | 倾斜角(X) | X方向倾斜，一侧近一侧远 |
| 倾斜(Y) | $h = y \cdot \tan\theta$ | 倾斜角(Y) | Y方向倾斜 |
| 柱面(X) | $h = C \cdot x^2$ | 曲率 | X方向弯曲，中心与边缘距离不同 |
| 柱面(Y) | $h = C \cdot y^2$ | 曲率 | Y方向弯曲 |
| 球面 | $h = C \cdot (x^2 + y^2)$ | 曲率 | 双向弯曲，模拟场曲 |
| 抛物面 | $h = C \cdot (x^2 + y^2)$ | 曲率 | 类球面，用于反射聚焦 |
| 锥面 | $h = C \cdot \sqrt{x^2 + y^2}$ | 曲率 | 径向线性变化，轴锥镜 |

### 参数说明

| 参数 | 说明 | 单位 | 典型值 |
|------|------|------|--------|
| 曲率 | 控制弯曲面的曲率大小 | 1/m | 正值凸起，负值凹陷 |
| 倾斜角(X) | X方向倾斜角度 | ° | -45° 到 45° |
| 倾斜角(Y) | Y方向倾斜角度 | ° | -45° 到 45° |
| 深度切片数 | 计算精度 | - | 5-30，越多越精确但越慢 |

### 显示标签页

1. **接收面形状**：高度图，显示接收面的三维形状（颜色表示高度）
2. **平面衍射**：标准平面接收面的衍射图样（参考基准）
3. **接收面衍射**：施加接收面形状后的衍射图样
4. **差异对比**：接收面衍射与平面衍射的差值

底部截面曲线对比图显示中心水平截面的强度分布，蓝色为平面参考，橙色为接收面结果。

### 使用方法

1. 从菜单 **仿真 → 接收面形状分析** 打开对话框
2. 选择接收面形状类型
3. 设置曲率或倾斜角参数
4. 调整深度切片数（精度与速度的权衡）
5. 点击"计算接收面衍射"
6. 在右侧标签页中查看不同视角的结果

### 注意事项

- **必须使用菲涅尔传播模型**，夫琅和费模型与距离无关，无法体现接收面形状的影响
- 曲率值过大可能导致有效距离为负（物理上不合理），程序会自动截断到最小值
- 倾斜角度较大时，有效距离变化范围大，需要更多深度切片以保证精度

## 光学场景预设

### 功能说明

光学场景预设提供 8 种典型光学实验场景，一键加载预设参数，方便快速开始专业模拟。

### 预设场景

| 场景 | 孔径 | 传播模型 | 说明 |
|------|------|---------|------|
| 望远镜衍射 | 圆形 5000μm | 夫琅和费 | 大口径远场衍射，观察艾里斑 |
| 反射望远镜 | 六边形 8000μm | 夫琅和费 | 六边形主镜衍射，模拟 JWST 等 |
| 显微镜成像 | 圆形 5μm | 菲涅尔 | 小孔径近场 + 球差 |
| 光纤模式 | 圆形 62.5μm | 菲涅尔 | 光纤芯径衍射模式 |
| 光谱仪光栅 | 光栅 200μm | 夫琅和费 | 多缝衍射分光 |
| 杨氏双缝 | 双缝 100μm | 夫琅和费 | 经典双缝干涉实验 |
| 单缝衍射教学 | 矩形 50μm | 夫琅和费 | sinc² 包络观察 |
| 圆环衍射 | 圆环 100μm | 夫琅和费 | 环形孔径，中央凹陷 |

### 使用方法

1. 从菜单 **场景 → 光学场景预设** 打开对话框
2. 在左侧列表选择场景
3. 右侧查看场景描述和参数配置
4. 点击"加载此场景"将参数应用到主界面
5. 主界面自动更新计算结果
